# 🌌 Nguyễn Thái Huy - AI Cyber-Tech Portfolio

## 🛠️ Tech Stack Arsenal
- **Languages:** Python, C, C++, JavaScript/Node.js
- **Framework:** React.js (Bootstrapped with Vite)
- **Styling:** Tailwind CSS (Custom Void Black, Neon Purple, Cyber Cyan themes)
- **Animation:** GSAP (ScrollTrigger, SplitText, QuickTo)
- **3D Rendering:** Three.js, @react-three/fiber, @react-three/drei
- **Tools:** Git, GitHub, MongoDB, Vercel
- **Icons:** react-icons

## ✨ Key Features
- **Interactive Neural Network:** A `<canvas>` background that draws connecting neural lines tracking the mouse at 60 FPS.
- **3D GPU Core:** A fully interactive, floating **RTX 5090** 3D model that reacts to mouse parallax.
- **Music Vibes Integration:** Embedded fully playable Spotify and SoundCloud tracks to set the portfolio's vibe.
- **Advanced GSAP Animations:**
  - `SplitText` typography reveals with Glitch effects.
  - Infinite Marquee for the Tech Stack showcase (slows down on hover).
  - Horizontally stacked scrolling for Projects using pinned `ScrollTrigger`.
  - High-intensity Magnetic Footer for social links.

## 💻 How to Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/nguyenthaihuy0913/nth-cs-portfolio.git
   cd nth-cs-portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to `http://localhost:5173`

## ☁️ How to Deploy (Vercel)
Deploying to **Vercel** is extremely easy and highly recommended.
1. Go to [Vercel](https://vercel.com/) and sign in with your GitHub account.
2. Click **Add New -> Project**.
3. Import this repository (`nth-cs-portfolio`).
4. Vercel will automatically detect that it's a Vite project. Click **Deploy**.
