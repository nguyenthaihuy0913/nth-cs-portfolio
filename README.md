## 🇺🇸 English

Welcome to my personal portfolio! This project is a premium, high-performance Single Page Application (SPA) designed with an **AI Cyber-Tech & Glassmorphism** aesthetic. It features heavy use of 3D rendering and advanced GSAP animations.

![Portfolio Preview](https://via.placeholder.com/1200x600/050511/00f3ff?text=AI+Cyber+Portfolio)

### 🚀 Live Demo
*https://nth-cs-portfolio.vercel.app/*

### 🛠️ Tech Stack Arsenal
- **Languages:** Python, C, C++, JavaScript/Node.js
- **Framework:** React.js (Bootstrapped with Vite)
- **Styling:** Tailwind CSS (Custom Void Black, Neon Purple, Cyber Cyan themes)
- **Animation:** GSAP (ScrollTrigger, SplitText, QuickTo)
- **3D Rendering:** Three.js, @react-three/fiber, @react-three/drei
- **Tools:** Git, GitHub, MongoDB, Vercel
- **Icons:** react-icons

### ✨ Key Features
- **Interactive Neural Network:** A `<canvas>` background that draws connecting neural lines tracking the mouse at 60 FPS.
- **3D GPU Core:** A fully interactive, floating **RTX 5090** 3D model that reacts to mouse parallax.
- **Music Vibes Integration:** Embedded fully playable Spotify and SoundCloud tracks to set the portfolio's vibe.
- **Advanced GSAP Animations:**
  - `SplitText` typography reveals with Glitch effects.
  - Infinite Marquee for the Tech Stack showcase (slows down on hover).
  - Horizontally stacked scrolling for Projects using pinned `ScrollTrigger`.
  - High-intensity Magnetic Footer for social links.

### 💻 How to Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/nguyenthaihuy0913/nth-cs-portfolio.git
   cd nth-cs-portfolio
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Start the development server:**
   ```bash
   npm run dev
   ```
4. **Open your browser:** Navigate to `http://localhost:5173`

### ☁️ How to Deploy (Vercel)
Deploying to **Vercel** is extremely easy and highly recommended.
1. Go to [Vercel](https://vercel.com/) and sign in with your GitHub account.
2. Click **Add New -> Project**.
3. Import this repository (`nth-cs-portfolio`).
4. Vercel will automatically detect that it's a Vite project. Click **Deploy**.

## 🇻🇳 Tiếng Việt

Chào mừng bạn đến với trang Portfolio cá nhân của tôi! Đây là một dự án Single Page Application (SPA) cao cấp, hiệu năng cao, được thiết kế theo phong cách **AI Cyber-Tech & Glassmorphism**. Giao diện nổi bật với các mô hình 3D thực tế và hiệu ứng tương tác GSAP mượt mà.

### 🚀 Xem Trực Tiếp (Live Demo)
*https://nth-cs-portfolio.vercel.app/*

### 🛠️ Kho Vũ Khí Công Nghệ (Tech Stack)
- **Ngôn ngữ lập trình:** Python, C, C++, JavaScript/Node.js
- **Framework:** React.js (Khởi tạo bằng Vite)
- **Thiết kế giao diện:** Tailwind CSS (Theme tùy chỉnh: Void Black, Neon Purple, Cyber Cyan)
- **Hiệu ứng Animation:** GSAP (ScrollTrigger, SplitText, QuickTo)
- **Môi trường 3D:** Three.js, @react-three/fiber, @react-three/drei
- **Công cụ:** Git, GitHub, MongoDB, Vercel
- **Icons:** react-icons

### ✨ Tính Năng Nổi Bật
- **Mạng Nơ-ron Tương Tác:** Nền background sử dụng `<canvas>` để vẽ các đường nối nơ-ron đuổi theo con trỏ chuột ở tốc độ 60 FPS.
- **Lõi Đồ Họa 3D:** Một mô hình 3D **RTX 5090** lơ lửng, hoàn toàn có khả năng tương tác xoay (parallax) theo hướng chuột của người dùng.
- **Âm Nhạc Kỹ Thuật Số (Music Vibes):** Nhúng trực tiếp trình phát nhạc Spotify và SoundCloud vào giao diện, có thể phát nhạc trực tiếp ngay trên web.
- **Hiệu Ứng GSAP Nâng Cao:**
  - `SplitText`: Hiệu ứng tách và xuất hiện từng chữ cái kết hợp Glitch.
  - Infinite Marquee: Dải băng chạy vô tận hiển thị kỹ năng (tự động giảm tốc độ khi lướt chuột qua).
  - Thanh trượt dự án ngang (Horizontal Scroll) xếp chồng lên nhau thông qua tính năng ghim (Pin) của `ScrollTrigger`.
  - Chân trang (Footer) thiết kế cực đại với hiệu ứng "Magnetic" hút từ tính mạnh dành cho các icon mạng xã hội.
