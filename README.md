## 🛠️ Tech Stack
- **Framework:** React.js (Bootstrapped with Vite)
- **Styling:** Tailwind CSS (Custom Void Black, Neon Purple, Cyber Cyan themes)
- **Animation:** GSAP (ScrollTrigger, SplitText, QuickTo)
- **3D Rendering:** Three.js, @react-three/fiber, @react-three/drei
- **Icons:** react-icons

## ✨ Key Features
- **Interactive Neural Network:** A `<canvas>` background that draws connecting neural lines tracking the mouse at 60 FPS.
- **3D GPU Core:** A fully interactive, floating **RTX 5090** 3D model that reacts to mouse parallax.
- **Advanced GSAP Animations:**
  - `SplitText` typography reveals.
  - Infinite Marquee for the Tech Stack showcase.
  - Horizontally stacked scrolling for Projects using pinned `ScrollTrigger`.
  - High-intensity Magnetic Footer for social links.

## 💻 How to Run Locally

1. **Clone the repository** (if you haven't already):
   ```bash
   git clone https://github.com/nguyenthaihuy0913/my_profile.git
   cd my_profile
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to `http://localhost:5173`

## ☁️ How to Deploy (Vercel - Recommended)
Since this project uses Vite, deploying to **Vercel** is extremely easy and highly recommended for the best performance.

1. Create a GitHub repository and push this code to it.
2. Go to [Vercel](https://vercel.com/) and sign in with your GitHub account.
3. Click **Add New -> Project**.
4. Import your repository.
5. Vercel will automatically detect that it's a Vite project. Just click **Deploy**!
